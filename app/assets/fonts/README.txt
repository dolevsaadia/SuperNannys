Download Heebo font files from Google Fonts:
https://fonts.google.com/specimen/Heebo

Place these files here:
- Heebo-Regular.ttf    (weight 400)
- Heebo-Medium.ttf     (weight 500)
- Heebo-SemiBold.ttf   (weight 600)
- Heebo-Bold.ttf       (weight 700)
- Heebo-ExtraBold.ttf  (weight 800)

Or run: flutter pub get (pubspec.yaml uses google_fonts package as fallback)
